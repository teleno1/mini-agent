# Full-Auto authorizes Code-Work Commands

Full-Auto will automatically authorize Code-Work Commands within the selected Workspace, including declared-dependency retrieval, so capability tests can measure an end-to-end coding loop rather than a non-interactive confirmation denial. The alternative of a separate experimental policy was rejected because Full-Auto is the product's explicit automation mode; deletion or overwrite, Git commit/push, publishing, credential access, arbitrary download-and-execute behavior, and Workspace escape remain outside its authority.
